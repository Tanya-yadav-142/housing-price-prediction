<h1 align="center">🏡 House Price Prediction</h1>

<p align="center">
End-to-End Machine Learning Project to Predict House Prices using Data Analysis and Regression Models
</p>

<hr>

<h2>📌 Project Overview</h2>
<p>
This project aims to predict house prices based on various features such as area, number of rooms, bathrooms, furnishing status, and amenities. 
It covers the complete machine learning pipeline including data cleaning, exploratory data analysis (EDA), feature engineering, model building, and evaluation.
</p>

</ul>

<h2>📊 Dataset Features</h2>

</ul>

<hr>

<h2>📌 Key Insights</h2>
<ul>
  <li>Area has the strongest impact on house price</li>
  <li>Bathrooms and number of rooms significantly influence price</li>
  <li>Air conditioning increases property value</li>
  <li>Unfurnished houses generally have lower prices</li>
</ul>

<hr>

<h2>📊 Model Performance</h2>
<p>
The Random Forest model performed better than Linear Regression, as it captures non-linear relationships and provides more accurate predictions.
</p>

<hr>

<h2>💡 Business Recommendation</h2>
<p>
Real estate businesses should prioritize properties with better amenities such as more bathrooms, air conditioning, and preferred locations to maximize property value and attract buyers.
</p>
